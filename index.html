A. Memahami HTML

Menurut Solichin (2016) HTML merupakan bahasa pemrograman dasar yang dikuasai oleh seseorang ahli web developer. Ini merupakan pondasi utama dalam pembuatan website. HTML dimanfaatkan untuk memformat halaman web dan disimpan dengan ekstensi .html. Halaman-halaman ini dapat diakses melalui browser, seperti Google Chrome, Mozilla Firefox, Opera, dan Safari.


Dalam pembuatan website, HTML dapat digabungkan dengan CSS, JavaScript, dan PHP untuk menciptakan halaman web yang dinamis. Halarnan yang dibuat hanya dengan HTML disebut halaman statis, yang tidak memiliki fungsi pengelolaan. Menurut Abdullah (2018), HTML terdiri atas elemen-elemen yang terdiri atas tag-tag, seperti heading, paragraf, form, tombol, daftar, dan hyperlink.
